# Screenshots Placeholder

This folder contains screenshot placeholders referenced in the main README.

## Required Screenshots

1. **overview.png** - System overview showing all components
2. **registration.png** - Public registration form
3. **tv-display.png** - Large format TV display screen
4. **mc-view.png** - MC/announcer interface
5. **staff-panel.png** - Staff control panel

## How to Add Screenshots

1. Take screenshots of each interface
2. Optimize images (recommended: 1920x1080 for displays, 800x600 for forms)
3. Save as PNG format
4. Name according to the list above
5. Place in this folder

## Screenshot Guidelines

- Use realistic test data (not "Test User" or "123456")
- Show the interface in use (with queue numbers, tickets, etc.)
- Ensure good lighting and contrast
- Crop to show only relevant UI elements
- Remove any sensitive information

## Tools for Screenshots

- **Windows**: Snipping Tool, Win + Shift + S
- **Mac**: Cmd + Shift + 4
- **Linux**: gnome-screenshot, Flameshot
- **Browser**: DevTools device toolbar for responsive views

## Image Optimization

Use tools like:
- TinyPNG (https://tinypng.com)
- ImageOptim (Mac)
- Squoosh (https://squoosh.app)

Target file size: < 500KB per image
